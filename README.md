# basic-website

Basic Nextjs react website

1. `brew install node` - `node -v`, `npm -v`
2. `npx create-next-app@latest app-name`
   - typescript: no
   - ESLint: optional
   - Tailwind: No
   - turbo:
   - App Router: yes
   - src/ directory: no
   - import alias: no
3. cd `app-name`
4. npm run dev
